## About ##

### Description ###

This project introduces basic data structures and algorithms that are used very often in everyday programming tasks to store and manipulate data efficiently.

The basic represented data structures are:

- Dynamic Arrays
- Linked Lists
- Stacks
- Queues
- Binary Trees
- Hash Tables
- Binary Search Trees
- Red Black Trees
- Graphs
